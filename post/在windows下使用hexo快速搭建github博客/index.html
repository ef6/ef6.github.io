在windows下使用hexo快速搭建github博客

配置环境

  • 安装Node.js

  • 安装Git终端

  • 申请GitHub账号

  • 创建名为<yourname>.github.io的仓库

  • Add SSH key

    • 右键桌面点击 Git Bash Here 输入 $ ssh-keygen -t rsa -C <Github注册邮箱> 生成密钥

    • 一路Enter，在 C:/Users/用户名/.ssh/ 下会生成两个文件 id_rsa（私钥）和 id_rsa.pub（公钥）

    • 记事本打开 id_rsa.pub 复制所有内容

    • 进入 Add SSH key 点击 New SHH key 粘贴到Key的输入框，Title里面的内容随便写，点击 Add SSH key

安装配置Hexo

继续使用bash会出现错误 hexo module.js:471 throw err; 因此使用cmd

安装

> npm install -g hexo-cli

初始化Hexo

> hexo init

打开 ./_config.yml 文件进行配置

# Site                 //网站相关信息
title: evan-fan
subtitle:
description:
author: evan-fan
language: zh-CN
timezone: Asia/Shanghai

# URL                  //配置统一资源定位符（个人域名）
url: http://evan-fan.github.io

# Deployment           //配置部署
deploy:
  type: git
  repo: https://github.com/evan-fan/evan-fan.github.io.git
  branch: master

其中repo项是之前Github上创建好的仓库的地址，形如

https://github.com/<yourname>/<yourname>.github.io.git

Hexo的使用

新建一篇文章

> hexo new <title> 打开 ./source/_post/<title>.md 进行编辑

生成静态页面

> hexo generate
or > hexo g

本地预览

> hexo server
or > hexo s
打开http://localhost:4000/进行预览

部署页面

> hexo deploy
or > hexo d
若出现 ERROR Deployer not found: git 时，在blog目录执行以下命令 > npm install hexo-deployer-git --save

生成并部署

> hexo d -g
